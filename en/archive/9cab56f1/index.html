What is a Domain Name?

Imagine the internet as a giant city, inhabited by countless websites and servers (where website content is stored). Each server has a unique "address," which is a string of numbers called an IP address.

For example, your home address might be "XX Province, XX City, XX District, XX Street, XX Number," while an IP address would be a string of numbers like "192.168.1.1" or "203.0.113.45".

But which is easier to remember: a long string of numbers or a name? Definitely a name! For example, if you want to go to Taobao, you'll remember "taobao.com," not a string of numbers.

A domain name is the name given to a server (or website) on the internet. It's like giving your website a memorable and distinctive name, making it easy for people to access.

Simply put: a domain name is the name of a website, used to replace a difficult-to-remember IP address.

What are the uses of a Domain Name?

The main uses of a domain name are:

  1. Easy to remember and access:

  2. Just as it's much easier to remember a friend's name than their house number, a domain name allows users to easily remember and access your website.

  3. Without a domain name, you would only be able to access websites via IP addresses, which is very inconvenient for ordinary users.

  4. Building brand image:

  5. A good domain name is like a catchy name that helps you build your brand image.

  6. For example, "baidu.com" represents the Baidu company, and "tencent.com" represents Tencent.
  7. A professional and easy-to-remember domain name leaves a good impression on users.

  8. Email addresses:

  9. Domain names can also be used to create professional email addresses, such as [email protected].

  10. This appears more professional and trustworthy than using free email services (such as @gmail.com or @qq.com).

  11. Unique identifier for a website:

  12. Domain names are unique on the internet. Once you register a domain name, no one else can register the exact same domain name.

  13. This ensures the uniqueness of your website on the internet. ---

How to Obtain a Domain Name (Taking Registering a .cn Domain on Alibaba Cloud as an Example)

Obtaining a domain name is like "registering" your website; you need to register it with a specialized institution (a domain registrar). Alibaba Cloud is a large domain registrar.

Let's take registering a .cn domain on Alibaba Cloud as an example. The steps are simple:

  1. Open the Alibaba Cloud official website:

  2. Enter www.aliyun.com in your browser to access the official Alibaba Cloud website.

  3. Search for the domain you want to register:

  4. On the Alibaba Cloud website's homepage or product page, find the "Domain" section.

  5. You will see a search box. Enter the domain name you want to register, for example, if you want to register "mywebsite.cn", enter mywebsite.cn.
  6. Click search.

  7. Check availability and add to cart:

  8. Alibaba Cloud will tell you if the domain name has already been registered by someone else.

  9. If it's not registered, it will show "Available for registration," and you can click "Add to cart."
  10. If it's already registered, it will show "Registered," and you will need to try a different name.

  11. Proceed to checkout and fill in information:

  12. Go to your shopping cart, confirm the domain name you want to register, and then click "Proceed to checkout."

  13. At this point, you need to fill in some registration information, mainly your personal information (name, ID number, contact information, etc.) or company information.
  14. Note: Registering a .cn domain usually requires real-name verification. You will need to upload a photo of your ID card or provide your company's business license information. This is a management regulation for domain names in mainland China.

  15. Pay the fee:

  16. After filling in the information, select a payment method and complete the payment. Domain prices vary depending on the suffix (such as .com, .cn, .net, etc.) and the registrar.

  17. Complete registration:

  18. After successful payment, Alibaba Cloud will complete the domain registration for you. This usually takes a few minutes to a few hours.

  19. Once registered, the domain name belongs to you, and you can manage it through the Alibaba Cloud management console.

In short: It's like naming your child; first, check if the name has already been used by someone else, if not, register it with the relevant department, pay a fee, and the name is yours.

What is Domain Name Resolution? Domain name resolution is like establishing a "connection" between your domain name (the name) and its IP address (the house number).

Imagine you have a friend named "Xiao Ming," who lives at "XX Community, Building XX, Unit XX, Apartment XX." You only know the name "Xiao Ming," but you don't know his exact address. Domain name resolution works like this: when you tell the postman the name "Xiao Ming," the postman knows to go to "XX Community, Building XX, Unit XX, Apartment XX" to find him.

On the internet, when you type a domain name (such as www.example.com) into your browser, your computer doesn't know the corresponding IP address. This is where the domain name resolution service comes in to tell your computer: the domain name www.example.com corresponds to the IP address 192.168.1.100.

Simply put: Domain name resolution tells the computer which IP address (house number) corresponds to the domain name (name) you entered.

How to Resolve Domain Names (Taking common A, AAAA, and CNAME records as examples)

Domain name resolution is achieved by setting DNS records in the domain registrar's management backend (such as Alibaba Cloud's domain management page). DNS records are like "address book entries" for your domain name.

The most common record types are:

1. A Record (Address Record)

  • Function: Points a domain name (or subdomain) to an IPv4 address.
  • Simple explanation: This is the most commonly used record. It's like telling the postman: "'Xiao Ming' lives at the address '192.168.1.100'."
  • Example:
  • You want www.example.com to point to your server's IP address 1.2.3.4.
  • In Alibaba Cloud's DNS management page, you would add a record:
  • Type: A
  • Host Record (or Subdomain): www (If you want to resolve example.com itself, enter @ or leave it blank)
  • Record Value (IP Address): 1.2.3.4

  • TTL (Time To Live): The default value is usually fine; it indicates how often this record is updated. #### 2. AAAA Record (IPv6 Address Record)

  • Function: Maps a domain name (or subdomain) to an IPv6 address.

  • Simple Explanation: Similar to A records, but IPv6 addresses are much longer than IPv4 addresses and represent the next generation of IP addresses. It's like telling the postman: "'Xiao Ming' lives at the address '2001:0db8:85a3:0000:0000:8a2e:0370:7334'."
  • Example:
  • You want www.example.com to point to your server's IPv6 address 2001:db8::1.
  • On the Alibaba Cloud DNS management page, you would add a record:
  • Type: AAAA
  • Host Record: www
  • Record Value (IPv6 Address): 2001:db8::1
  • TTL: Default value

3. CNAME Record (Canonical Name Record)

  • Function: Maps a domain name (or subdomain) to another domain name.
  • Simple Explanation: This is like telling the postman: "'Xiao Ming's' house is actually 'Lao Wang's' house." In other words, you're giving this domain name an "alias".
  • Example:
  • You have a service, such as a CDN (Content Delivery Network), which provides you with a fixed domain name, such as mycdn.aliyuncs.com.
  • You want your domain name blog.example.com to point directly to this CDN service.
  • On the Alibaba Cloud DNS management page, you would add a record:
  • Type: CNAME
  • Host Record: blog
  • Record Value (Target Domain Name): mycdn.aliyuncs.com
  • TTL: Default value

Important Note:

  • Effective Time: After modifying DNS records, it usually takes a few minutes to tens of minutes (sometimes even longer, up to 24 hours) for the changes to take effect globally. This process is called "DNS propagation". * Host Record:
  • @ or leave blank: Represents the domain itself, such as example.com.
  • www: Represents www.example.com.
  • blog: Represents blog.example.com.
  • mail: Represents mail.example.com.

Summary of the resolution process:

  1. You register a domain name example.com.
  2. You purchase a server with an IP address 1.2.3.4.
  3. In the Alibaba Cloud domain management console, you add an A record for example.com, with the host record being @ and the record value being 1.2.3.4.
  4. When someone enters example.com in their browser, their computer queries the DNS server for the IP address of example.com. The DNS server tells them it's 1.2.3.4.
  5. Knowing the IP address, the computer can connect to your server and display your website content.
