隐私政策

生效日期：2026年01月01日

感谢您访问 Cnfte Space（网站地址：https://cnfte.top，以下简称“本网站”）。我们深知隐私保护对您的重要性，致力于保障您的个人信息安全与合法权益。本隐私政策旨在明确告知您本网站（静态站点，使用 Cloudflare CDN 加速，未绑定数据库，已加入 Google Adsense 广告计划）如何收集、使用、存储、保护您的个人信息，以及您享有的相关权利。请您在使用本网站服务前仔细阅读并理解本政策，您的访问、浏览或使用本网站服务的行为，即表示您已同意本隐私政策的全部内容。

一、适用范围

本隐私政策适用于您访问本网站、使用本网站提供的所有服务（包括但不限于技术博客浏览、资源分享、网络协议科普内容查阅、广告浏览等）过程中，本网站及相关第三方服务提供商（包括 Cloudflare、Google Adsense 等）收集、使用、存储和保护您个人信息的所有行为。 本政策不适用于除上述第三方服务提供商外的其他第三方网站或服务，本网站可能包含指向第三方网站的链接（如 GitHub 资源链接等），您访问第三方网站时，其隐私政策与本网站无关，我们建议您仔细阅读第三方的隐私政策。

二、收集的信息类型及方式

本网站为静态站点且未绑定数据库，不主动存储您的个人身份信息，收集的信息主要通过自动方式获取，具体如下：

（一）本网站自身自动收集的信息

当您访问和使用本网站时，我们可能通过服务器日志等技术自动收集以下非个人身份信息，此类信息不直接关联您的真实身份，且不会存储在本网站服务器（因无绑定数据库）： 1. 设备信息：包括您使用的设备类型、操作系统版本、浏览器类型、设备标识符（如 IMEI、MAC 地址）等； 2. 访问信息：包括您的访问时间、访问时长、访问的页面路径、浏览的内容、点击记录、IP 地址（经 Cloudflare 处理后可能为匿名化 IP）、网络运营商、所在地区等； 3. 技术日志信息：包括服务器响应时间、错误日志、访问请求头等技术数据。

（二）第三方服务收集的信息

  1. Cloudflare CDN 相关收集：为实现网站加速、安全防护等功能，本网站使用 Cloudflare CDN 服务。Cloudflare 可能会收集您访问本网站时的相关信息，包括但不限于 IP 地址、浏览器类型、访问时间、页面请求等，用于节点调度、攻击防护、访问统计等目的。Cloudflare 的数据处理遵循其自身的隐私政策（详见 Cloudflare 官方隐私政策页面）；
  2. Google Adsense 广告相关收集：本网站已加入 Google Adsense 广告计划，为向您展示相关且个性化的广告，Google Adsense 可能会收集您的浏览行为、设备信息、IP 地址、兴趣偏好等信息。Google Adsense 的信息收集与使用遵循 Google 的隐私政策及广告服务条款（详见 Google 官方隐私政策及 Adsense 服务条款页面）。Google 可能会使用 Cookie 或类似追踪技术收集上述信息，具体可参考 Google 关于 Cookie 的相关说明。

（三）您主动提供的信息

仅在您自愿选择参与特定互动功能或服务时，我们可能收集您主动提供的相关信息，例如： 1. 若您选择通过网站留言、评论等第三方嵌入功能（如第三方评论插件）分享观点或咨询问题，相关信息将由对应的第三方服务提供商按照其隐私政策收集和处理，本网站不存储此类信息； 2. 若您参与本网站组织的资源分享、技术交流等活动，根据活动要求可能收集您的相关必要信息（如用于资源获取的验证信息等），具体以活动规则为准，且相关信息仅用于活动相关用途，活动结束后将按规定删除或匿名化处理。

三、信息的使用目的

（一）本网站层面的使用

我们收集的非个人身份信息仅用于以下合法目的： 1. 提供和维护网站服务：确保本网站正常运行，结合 Cloudflare CDN 优化网站加载速度和访问稳定性，为您提供流畅的浏览体验； 2. 内容优化：根据访问数据（如页面浏览量、点击热度等）分析用户偏好，优化网站内容布局、更新频率，为您提供更贴合需求的技术博客、科普内容及资源； 3. 安全保障：结合 Cloudflare 的安全防护能力，分析访问日志，识别和防范恶意攻击、网络欺诈等安全风险，保护您和本网站的合法权益； 4. 合规要求：遵守相关法律法规、监管要求，配合有权机关的合法调查。

（二）第三方服务的使用目的

  1. Cloudflare 对收集信息的使用：主要用于提供 CDN 加速、DDoS 防护、访问路由优化等服务，具体以 Cloudflare 隐私政策为准；
  2. Google Adsense 对收集信息的使用：主要用于分析用户兴趣偏好，展示个性化广告，统计广告曝光量、点击量等数据，以便向本网站提供广告收益结算服务，具体以 Google 隐私政策及 Adsense 服务条款为准。

四、信息的存储与保护

（一）本网站的存储与保护

因本网站为静态站点且未绑定数据库，不存储任何用户个人信息（包括主动提供的信息及自动收集的信息），所有自动收集的临时访问数据仅用于实时访问分析及安全校验，不会留存。

（二）第三方服务的存储与保护

  1. Cloudflare 存储与保护：您的相关访问信息将由 Cloudflare 按照其数据存储政策存储在其安全服务器中，Cloudflare 采用符合行业标准的安全技术措施（如数据加密、访问权限控制等）保护信息安全，存储期限以 Cloudflare 隐私政策为准；
  2. Google Adsense 存储与保护：Google Adsense 收集的信息将由 Google 按照其数据存储规范存储，Google 会采取严格的安全保护措施保障信息安全，存储期限以 Google 隐私政策为准。

（三）共同安全保障责任

我们已审慎选择具备良好隐私保护资质的第三方服务提供商（Cloudflare、Google Adsense），并通过服务协议明确要求其遵守相关法律法规，采取必要的安全保护措施，不得擅自泄露、篡改、滥用收集的信息。 尽管已采取上述措施，但互联网环境存在不确定性，我们无法完全保证第三方服务提供商的信息传输与存储绝对安全。您应注意妥善保管自己的设备信息及网络访问环境，避免因自身操作不当导致信息泄露。

五、信息的共享与披露

（一）本网站的信息共享与披露

本网站不存储任何用户个人信息，因此不存在向第三方共享、出售、出租用户个人信息的行为。仅在符合以下情况时，可能配合第三方披露相关非个人身份信息（如汇总后的访问统计数据）： 1. 获得您的明确同意； 2. 法律法规要求或司法裁定、行政命令强制要求； 3. 为保护本网站、您或其他第三方的合法权益，在合理且必要的范围内。

（二）第三方服务的信息共享与披露

Cloudflare、Google Adsense 等第三方服务提供商对信息的共享与披露，遵循其自身的隐私政策及相关法律法规，本网站不干预其合法的信息处理行为，但会要求其不得超出服务必要范围共享用户信息。

六、Cookie 及类似追踪技术的使用

（一）本网站的 Cookie 使用

本网站自身不主动设置 Cookie，但为实现网站功能优化，可能会通过第三方服务（如 Cloudflare、Google Adsense）使用 Cookie 或类似追踪技术（如网页信标、设备标识符等）。

（二）第三方服务的 Cookie 使用

  1. Cloudflare 可能会使用 Cookie 存储您的访问偏好、验证访问状态等，以实现 CDN 加速和安全防护功能；
  2. Google Adsense 可能会使用 Cookie 收集您的浏览行为、兴趣偏好等，以展示个性化广告。 您可以通过浏览器设置禁用或删除 Cookie，具体操作方式可参考浏览器的帮助文档。若禁用 Cookie，可能会影响本网站的加载速度、广告展示效果，或导致部分第三方服务功能无法正常使用。

七、您的权利

您对自己的个人信息享有以下权利，我们将协助您向相关第三方服务提供商主张权利（因本网站不存储信息）： 1. 访问权：您有权查询 Cloudflare、Google Adsense 等第三方收集的关于您的个人信息，可通过第三方官方渠道提交访问请求； 2. 更正权：若您发现第三方收集的您的个人信息存在错误或不完整，有权通过第三方官方渠道要求更正或补充； 3. 删除权：在符合法律法规及第三方隐私政策规定的情况下，您有权通过第三方官方渠道要求删除您的个人信息； 4. 撤回同意权：您有权撤回对第三方服务收集、使用您个人信息的同意（如通过 Google 账号设置关闭 Adsense 个性化广告授权），但撤回同意后可能影响相关服务功能的使用； 5. 投诉建议权：若您对本网站的隐私保护工作或第三方服务的信息处理行为有疑问、投诉或建议，可通过本政策预留的联系方式与我们沟通，我们将尽力协助您处理。

八、未成年人保护

本网站主要面向具备完全民事行为能力的用户，不专门针对未成年人提供服务。若未成年人未经监护人同意使用本网站服务，本网站及第三方服务提供商（Cloudflare、Google Adsense）不会主动收集未成年人的个人信息。如发现未成年人的个人信息被意外收集，我们将在接到监护人通知后，协助联系相关第三方服务提供商及时删除相关信息。建议监护人引导未成年人正确使用互联网，保护其个人信息安全。

九、隐私政策的更新

本网站可能根据法律法规的修订、网站服务的调整（如第三方服务变更）或隐私保护需求的变化，对本隐私政策进行更新。更新后的隐私政策将在本网站显著位置公示（如首页公告、单独的隐私政策更新页面等），并以修改日期标注生效时间。 您应定期查看本隐私政策的更新内容，若您在隐私政策更新后继续使用本网站服务，即表示您同意更新后的隐私政策；若您不同意，应立即停止使用本网站服务。

十、联系方式

若您对本隐私政策有任何疑问、投诉或建议，或需要协助行使相关权利，请通过以下方式与我们联系： 联系邮箱：cnfte.2@gmail.com

我们将在收到您的反馈后，在合理期限内与您沟通并协助处理相关事宜。
